# 2026北京车展信息网站设计文档

## 项目概述

本项目是一个专门用于搜集和展示2026年北京车展信息的网站，包含展会概览、车展对比、参展车型和发布会安排等内容。

## 核心原则

### 信息来源原则
**所有信息都需要有明确的数据和信息来源，不允许杜撰**

- 信息来源必须是权威的汽车媒体、官方发布渠道或行业认可的数据源
- 所有数据和信息需要在网站中或代码注释中注明来源
- 定期更新信息，确保数据的时效性和准确性
- 对于不确定的信息，标注为"预计"或"待公布"，避免误导用户

### GitHub操作原则
**不要主动推送项目到GitHub**

- 仅在明确要求时才进行GitHub推送操作
- 推送前确保所有信息都已验证和更新
- 推送时添加清晰的 commit 信息，说明更新内容

## 网站结构

1. **展会概览** - 包含展会主题、时间、场地等基础信息
2. **车展对比** - 对比2024年北京车展、2025年上海车展、2026年北京车展的基础数据
3. **参展车型** - 详细列出2026年北京车展的参展车型及其信息
4. **参展供应商** - 列出主要参展供应商及其主营业务
5. **发布会安排** - 展示车展期间的发布会信息

## 技术架构

- **前端技术**：HTML5, CSS3, JavaScript
- **数据管理**：前端静态数据
- **部署方式**：静态网站

## 数据更新流程

1. 定期搜索网络获取最新的车展信息
2. 验证信息来源的可靠性
3. 更新网站内容
4. 推送到GitHub仓库
